# Склад Азот: Собирай Заказы!

2D браузерный платформер про работницу склада Азот, котора собирает заказы, уворачивается от чужих тележек и старается набрать как можно больше очков до конца смены.

## О проекте

Игра сделана на чистом HTML, CSS и JavaScript без сборщиков и внешних библиотек.  
Проект запускается как статический сайт и подходит для размещения на Netlify, GitHub Pages или для локального запуска через браузер.

## Геймплей

Игрок перемещается по платформам склада, собирает заказы и уклоняется от тележек.

В игре есть несколько типов объектов:

- обычный заказ — даёт 10 очков
- срочный заказ — даёт 20 очков
- хрупкий заказ — даёт 30 очков, но может разбиться
- энергетик — временный бонус на ускорение и двойной прыжок

Сложность постепенно растёт по времени:
- появляется больше заказов
- увеличивается доля сложных заказов
- тележки появляются чаще
- тележек на уровне становится больше

## Основные механики

- платформенный геймплей с несколькими уровнями по высоте
- спуск сквозь платформы вниз
- переход через левый и правый край экрана
- 3 жизни на смену
- штраф за падение вниз уровня: -10 очков и -1 жизнь
- завершение смены после третьего падения
- штрафы за столкновения с тележками и потерю заказов
- таблица лидеров на `localStorage`
- тестовый режим по имени `tester`
- настройка звука
- настройка размера шрифта интерфейса

## Управление

- `A / D` или `← / →` - движение
- `W` или `↑` - прыжок
- `S` или `↓` - спуск сквозь платформу
- `Space` или `P` - пауза

## Тестовый режим

Если ввести имя `tester`, игра запустится в тестовом режиме.

Особенности тестового режима:
- результат не сохраняется в таблицу лидеров
- при падении вниз игрок не завершает смену окончательно

## Назначение файлов

* `index.html` - разметка экранов игры
* `css/style.css` - стили интерфейса, меню и экранов
* `js/main.js` - управление экранами, HUD, запуском игры и таблицей лидеров
* `js/game.js` - основная игровая логика, физика, объекты, тележки, платформы, бонусы
* `js/storage.js` - сохранение результатов и настроек в `localStorage`
* `js/audio.js` - управление звуком
* `assets/` - спрайты, фоновые изображения и графика меню

## Запуск

### Вариант 1. Локально

Открыть `index.html` в браузере.

### Вариант 2. Через Live Server

Если браузер ограничивает локальный запуск файлов:

1. открыть проект в VS Code
2. установить расширение Live Server
3. запустить `index.html` через Live Server

### Вариант 3. Через Netlify

Открыть по ссылке https://azot-game.netlify.app/

## Деплой

Проект является статическим, поэтому его можно разместить:

* на Netlify
* на GitHub Pages
* на любом другом статическом хостинге

## Особенности реализации

* игра работает полностью на клиенте
* результаты сохраняются в браузере пользователя
* никакой серверной части не требуется
* все игровые данные и интерфейс работают на JavaScript
