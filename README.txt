Азот: Сборщик заказов

Состав проекта:
- index.html
- css/style.css
- js/storage.js
- js/audio.js
- js/game.js
- js/main.js
- assets/sprites.png
- assets/bg-tile.png

Что исправлено:
- снижена сложность прыжков между верхними платформами
- таблица рекордов теперь показывает топ-9 и место игрока вместо 10-й строки, если игрок не попал в десятку
- исправлено сохранение ранга при одинаковых имени и очках

Как запустить:
1. Открыть index.html в браузере.
2. Если браузер блокирует локальный запуск, открыть проект через Live Server в VS Code.

Управление:
- A / D или стрелки влево / вправо - движение
- W или стрелка вверх - прыжок
- Space или P - пауза

Особенности:
- имя tester включает тестовый режим
- A- / A+ меняют размер шрифта
- результаты сохраняются в localStorage
- звук можно включать и выключать
- кнопка "Сдать смену" завершает текущий запуск вручную
